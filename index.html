Featured Project

PDLite is a prototype device for the early screening of Parkinson's disease using saliva biomarkers. It combines microfluidics, ELISA, OpenCV, and Python to analyze key biomarkers from a 1–3 mL saliva sample. Results are processed by a Raspberry Pi and accessed via a QR code on the PDLite website, providing risk assessment for users and detailed biomarker analysis for healthcare professionals.
